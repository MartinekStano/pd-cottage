# Project Brief

## Základné fakty

- Názov: Pallov Dvor
- Typ objektu: súkromná chata na krátkodobé ubytovanie
- Lokalita: Lutiše, Ulica 373
- Kapacita: až 16 osôb
- Rozdelenie: 2 samostatné ubytovacie jednotky
- Kapacita jednej jednotky: max. 8 osôb
- Parkovanie: 4 parkovacie miesta priamo pri objekte

## Charakter webu

Web má pôsobiť ako profesionálna prezentácia pokojnej súkromnej chaty. Má komunikovať súkromie, komfort, prírodu, oddych, dôveru a kultivovaný pobyt.

Nemá pôsobiť ako hotelový portál, penzión, reštauračný web ani web pre hlučné oslavy.

## Cieľ

Hlavný cieľ webu je priviesť návštevníka k nezáväznému overeniu dostupnosti termínu.

Primárne CTA:

- Overiť dostupnosť termínu

Sekundárne CTA:

- Pozrieť ubytovanie
- Pozrieť cenník
- Poslať nezáväzný dopyt
- Kontaktovať nás
- Zobraziť v navigácii

## Cieľové skupiny

- Rodiny s deťmi
- Páry
- Dospelí hostia
- Menšie skupiny priateľov, ktoré hľadajú pokojný pobyt
- Menšie firemné kolektívy

## Navigácia

- Domov
- O nás
- Ubytovanie
- Cenník
- Atrakcie
- Wellness
- Kontakt

V hlavičke má byť samostatné výrazné CTA `Overiť termín`.
